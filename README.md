# Project Voxel

A 2D sandbox game built in Unity inspired by games like Terraria. Project Voxel features procedurally generated worlds with dynamic terrain, cave systems, and ore distribution — all built from scratch using C# and Unity's game engine.

🎮 **[Play the Live Demo](https://gtschaaf.github.io/Project-Voxel-Web/)** | 📊 **[Code Review Presentation](https://docs.google.com/presentation/d/1mdhIo9wAQ7qO9RTO4M800Psf1unr7tE9rO5-6YxBe1k/edit?usp=sharing)**

---

## 🎮 About the Game

Project Voxel is a 2D sandbox experience where players explore an infinite, procedurally generated world. The game features destructible terrain, underground cave networks, and scattered ore deposits to discover. Custom pixel art, animations, and physics-based interactions bring the world to life.

---

## ✨ Features

- **Procedural World Generation** — Terrain generated using Perlin noise for natural, varied landscapes across every playthrough
- **Cave Systems** — Underground cave networks generated procedurally for exploration
- **Ore Generation** — Distributed ore deposits embedded throughout the world using noise-based placement
- **Physics Interactions** — Responsive physics systems for player and world objects
- **Custom Pixel Art** — Original sprites, tiles, and animations created for the game
- **Interactive UI** — In-game UI elements including inventory and world interaction systems

## 🚀 Getting Started

**Just want to play?** Click the [Live Demo](https://gtschaaf.github.io/Project-Voxel-Web/) link above — no installation needed, runs directly in your browser.

**Want to explore the source code locally?**

> Requires Unity to be installed.

1. Clone the repository
   ```bash
   git clone https://github.com/matthewfries99/project-voxel.git
   ```
2. Open the project in **Unity Hub**
3. Select the correct Unity version if prompted
4. Press **Play** in the Unity Editor to run the game
